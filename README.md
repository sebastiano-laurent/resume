# CV Web - Sebastiano Laurent

## Avvio locale
Da questa cartella:

```bash
cd /Users/sebastiano/Documents/Lavoro/CV_Perfetto/06_web_cv
python3 -m http.server 8080
```

Apri: [http://localhost:8080](http://localhost:8080)

## Personalizzazione rapida
- Contenuti IT/EN: `script.js` (oggetto `cvData`)
- Stile: `styles.css`
- Struttura pagina: `index.html`

## Export PDF
Usa il pulsante `Esporta PDF` (richiama `window.print()`).

## Pubblicazione
- GitHub Pages: push su repo e abilita Pages dalla cartella root o branch dedicato.
- Netlify/Vercel: deploy diretto della cartella `06_web_cv`.
