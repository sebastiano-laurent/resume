# Checklist CV Perfetto

- Ruolo target chiaro nel titolo
- Summary orientato al valore
- Esperienze con metriche e risultati
- Competenze allineate all'annuncio
- Keyword ATS presenti naturalmente
- Formato pulito (1 pagina quando possibile)
- Versione IT e EN pronte
- Link LinkedIn/GitHub aggiornati
